## 方法论说明

### 模型：Strength-Adjusted Bayesian Poisson（强度调整贝叶斯泊松）

**核心参数**：
- shrinkage = 1.5（给球队个体数据更多权重）
- 全局场均进球：2.99（小组赛72场215球）
- 淘汰赛修正：基于已完赛32强赛和16强赛数据动态调整

**强度计算公式**：
```
进攻强度 = 总进球 / 场次
防守强度 = 总失球 / 场次
净强度 = 进攻强度 - 防守强度
```

**预期进球计算**：
```
lambda_home = (1.5 × 全局场均 + 3 × home_进攻) / (1.5 + 3) - away_防守校正
lambda_away = (1.5 × 全局场均 + 3 × away_进攻) / (1.5 + 3) - home_防守校正
```
