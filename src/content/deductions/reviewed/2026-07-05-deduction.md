## 方法论说明

### 模型：Strength-Adjusted Bayesian Poisson

本推演使用分层强度模型，主要改进：

1. **球队强度计算**：从小组赛数据计算进攻/防守强度
2. **收缩参数**：shrinkage=1.5，给个体数据更多权重
3. **分档预期**：根据强度差定义三个档位（碾压局/优势局/均势局）
4. **校准数据窗口**：使用所有已完赛数据进行动态校准
5. **主场优势调整**：根据球队和场地位置进行调整

### 计算公式

```
lambda_home = (1.5 * 全局场均进球 + 3 * home_进攻强度) / (1.5 + 3) - away_防守强度校正 + 主场调整
lambda_away = (1.5 * 全局场均进球 + 3 * away_进攻强度) / (1.5 + 3) - home_防守强度校正
```

### >2.5球概率输出规则

概率根据强度差动态调整：
- 强弱分明的碾压局：0.60-0.78
- 正常优势局：0.40-0.60
- 均势或保守局：0.25-0.45
- 极端低分局：0.15-0.35
